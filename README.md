# Parking OCR API

FastAPI-based OCR microservice for extracting vehicle license plates using EasyOCR.

## Features

- License plate OCR
- FastAPI REST API
- JSON responses
- Ionic frontend support
- Cross-platform mobile integration

## Tech Stack

- Python
- FastAPI
- EasyOCR
- OpenCV
- Ionic Angular

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
